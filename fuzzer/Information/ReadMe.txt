No arguments!
In order to submit arguments, you must: 1) Open the CMD terminal.
2) Enter the full path to the file (C: \ ... \ ... \ Generation.exe)
3) Set the arguments, the program takes 4 arguments (C: \ ... \ ... \ Generation.exe first second third fourth)
FIRST - program name.
SECOND - path to in(prc) files.
THIRD - path to out(doc) files.
FOURTH - program execution code (1 - random generation of prc file, 2 - source test prc files are available)
4) Press the "Enter" key
